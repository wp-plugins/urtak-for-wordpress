== Description ==

Urtak is the best way for your visitors to respond to content. Ask and answer questions creating a structured conversation. The plugin allows you to kick off a conversation as an author by asking a few questions, from there users can contribute their own questions.

https://urtak.com/

== Installation ==

Upload the Urtak plugin to your blog, Activate it and follow the instructions.
